This skill provides comprehensive graphic design capabilities for the catering and restaurant industry. Use this when you need to create:

- **Posters & Flyers**: In-store displays, promotional materials, event announcements
- **Menus & Price Lists**: Restaurant menus, beverage lists, seasonal specials
- **Social Media Graphics**: WeChat moments, Xiaohongshu (RED), Instagram, Douyin covers
- **Branding Materials**: Business cards, takeout packaging, gift cards, stickers
- **Event Promotions**: Holiday specials, themed events, seasonal campaigns

## Catering Design Thinking

Before creating catering designs, understand the context:

### Key Questions

1. **Business Type**: Fine dining, casual cafe, fast food, bar/pub, bakery, tea house?
2. **Target Audience**: Young professionals, families, students, tourists, locals?
3. **Brand Personality**: Luxury, cozy, modern, traditional, playful, minimalist?
4. **Platform/Medium**: Print (A4/A3), Social Media (9:16, 1:1, 4:3), Digital menu?
5. **Purpose**: Promote new dish, announce event, daily special, brand awareness?
6. **Style Reference**: Any preferred styles or competitor references?

### Design Process

1. **Analyze**: Extract business type, cuisine, atmosphere, target audience
2. **Style**: Recommend appropriate design style (vintage, modern, minimal, etc.)
3. **Color**: Select palette matching brand and cuisine type
4. **Typography**: Choose fonts reflecting personality (elegant, playful, etc.)
5. **Layout**: Structure based on content priority and platform requirements
6. **Generate**: Create prompt for nano banana or other image generator
7. **Refine**: Iterate based on feedback

### 1. Posters & Flyers (海报/宣传单)

**Use Cases:**
- In-store announcements and promotions
- Event flyers for distribution
- Seasonal campaign visuals
- New dish/special introduction
- Holiday/celebration decorations

**Specifications:**
| Type | Size | Resolution | Format |
|------|------|------------|--------|
| A4 Poster | 2480×3508px | 300 DPI | PNG/PDF |
| A3 Poster | 3508×4961px | 300 DPI | PNG/PDF |
| Square Social | 1080×1080px | 72/300 DPI | PNG/JPG |
| Story Size | 1080×1920px | 72 DPI | PNG/JPG |

**Design Elements:**
- Strong headline with appetite appeal
- High-quality food photography or illustration
- Clear CTA (Limited Time, 20% Off, etc.)
- Brand elements (logo, colors, contact)

**Example Prompt Structure:**
```
[Style: Vintage/Modern/Minimal] restaurant poster for [dish/event]
featuring [main visual: grilled salmon/happy hour/NY party]
with color scheme: [warm oranges/cools blues/elegant blacks]
typography: [bold headlines/elegant serif]
mood: [cozy/energetic/sophisticated]
```

## AI Image Generation Guidelines

### Prompt Engineering Tips

1. **Be Specific About Style**: Include reference styles (e.g., "Vintage 1950s diner", "Modern Scandinavian")
2. **Define Color Palette**: Mention specific colors or color moods (e.g., "warm amber tones", "cool ocean blues")
3. **Specify Lighting**: "Natural daylight", "Warm candlelight", "Dramatic spotlight"
4. **Include Text Instructions**: Clearly state any text that should appear
5. **Reference Quality**: "Professional food photography", "Commercial quality", "High resolution"
6. **Consider Platform**: Match aspect ratio to intended use

### Platform-Specific Optimization

| Platform | Aspect Ratio | Style Notes |
|----------|-------------|-------------|
| WeChat Moment | 4:5 (1080×1350) | Clean, not too busy |
| Xiaohongshu | 1:1 or 9:16 | Aesthetic, lifestyle |
| Douyin | 9:16 | Bold, quick impact |
| Instagram Post | 1:1 (1080×1080) | Polished, cohesive |
| Instagram Story | 9:16 (1080×1920) | Full-screen, engaging |
| Print (A4) | 1:1.414 | High detail, 300 DPI |

### Resolution Guidelines

| Use Case | Resolution | Format |
|----------|------------|--------|
| **Social Media** | 72-150 DPI | PNG/JPG |
| **Digital Menu Screen** | 72-150 DPI | PNG/JPG/PDF |

## Pre-Delivery Checklist

Before delivering catering designs, verify:

### Visual Quality
- [ ] High-quality food photography or illustrations
- [ ] Appetizing colors that stimulate appetite
- [ ] Consistent brand elements throughout
- [ ] No grammatical errors in text
- [ ] Proper spelling of all items

### Text & Typography
- [ ] Clear hierarchy (title > subtitle > body)
- [ ] Readable at intended viewing distance
- [ ] Appropriate font sizes for format
- [ ] Consistent font usage
- [ ] Special characters handled correctly (e.g., ¥, °)

### Platform Requirements
- [ ] Correct aspect ratio for intended use
- [ ] Sufficient resolution (72 DPI for web, 300 DPI for print)
- [ ] Safe margins for text/content
- [ ] Color mode appropriate (RGB for digital, CMYK for print)

### Legal & Compliance
- [ ] No copyrighted images used without license
- [ ] Allergen information included where required
- [ ] Nutritional claims accurate
- [ ] Promotional terms clear and legal
- [ ] Contact information accurate

### Accessibility
- [ ] Sufficient contrast for readability
- [ ] Alt text provided for digital use
- [ ] Clear visual hierarchy
- [ ] Color not sole information carrier

## Quick Poster Generator (海报生成器)

### New Script: `generate_poster.py`

Generate promotional posters with dish name, prices, logo, and images.

```bash
# Interactive mode (recommended)
python3 scripts/generate_poster.py --interactive

# Command line mode
python3 scripts/generate_poster.py "招牌牛排" 198 168 --logo logo.png --dish-image steak.jpg

# Generate all 5 styles
python3 scripts/generate_poster.py "新品披萨" 88 68 --all-templates --promo-badge hot

# Open nano banana directly
python3 scripts/generate_poster.py "麻辣火锅" 128 98 --logo logo.png --open-nano
```

**Features:**
- Combines dish name, prices (regular + member), logo, and dish images
- 5 design styles: Modern, Elegant, Playful, Vintage, Minimal
- Automatic discount calculation
- Promotion badges (SALE, NEW, HOT, MEMBER, LIMITED)
- Generates HTML posters ready for viewing/printing
- Creates nano banana prompts for AI image generation

**Input Parameters:**
| Parameter | Description | Example |
|-----------|-------------|---------|
| `dish_name` | Dish name | "招牌牛排" |
| `price` | Regular price | 198 |
| `member_price` | Member price (optional) | 168 |
| `--logo` | Restaurant logo file path | logo.png |
| `--dish-image` | Dish photo file path | steak.jpg |
| `--promo-badge` | Promotion badge style | hot |
| `--description` | Dish description | "选用进口牛肉" |

**Output:**
- HTML poster files (printable)
- Text prompts for nano banana
- Saved templates for reuse
