## File Format Guidelines

| Format | Best For | Pros | Cons |
|--------|----------|------|------|
| **PNG** | Digital, transparency | Lossless, transparent | Larger file size |
| **JPG** | Photos, social | Compressed, small | Lossy compression |
| **PDF** | Print, multi-page | Vector/text support | Requires reader |
| **TIFF** | Professional print | High quality | Large files |
| **SVG** | Logos, icons | Vector, scalable | Not for photos |

## Color Mode Guidelines

| Use Case | Color Mode | Notes |
|----------|-----------|-------|
| **Print** | CMYK | Convert from RGB |
| **Digital/Web** | sRGB | Standard web color |
| **Social Media** | sRGB | Most platforms use |
| **Apps** | Display P3 | iOS support |
