# Format Research: What DevOps People Actually Buy (2025-2026)
**Date**: 2026-02-08
**Domain**: DevOps / Programming / Tech Education
**Key Finding**: PDFs are commoditized, not dead. The winning format at $29-79 is a "Decision Kit" -- not a static PDF.

## Executive Summary

People are not paying for static PDFs -- they are paying for **hands-on access, curated systems, and shortcut-to-competence bundles**. The winning formats combine practical, do-it-yourself components with expert curation and decision-making frameworks.

## What DevOps Engineers Actually Use to Learn

1. **GitHub repos with real infrastructure code + explanatory READMEs** -- Kelsey Hightower's "Kubernetes the Hard Way" (25k+ stars). The archetype: step-by-step markdown in a GitHub repo.
2. **Interactive labs** -- KodeKloud ($15-29/mo, 1M+ learners), iximiuz Labs ($250 lifetime, solo indie creator)
3. **Video + hands-on demos (not lectures)** -- Adrian Cantrill: 20-min max lessons, real AWS accounts
4. **Text + diagrams (not video)** -- ByteByteGo ($59.99/yr, 400k+ subscribers): "Complex ideas broken down using high-resolution charts and drawings"
5. **Paid newsletters** -- The Pragmatic Engineer ($15/mo, $1.5M+/yr revenue, 400k subscribers)

## The Anti-PDF Argument

**Ebooks are not dead, but they are commoditized:**
- Ebooks remain top 3 on Gumroad
- One creator makes $500/week from a single ebook
- BUT: a DevOps creator earned only $16 from their first Gumroad ebook sale

**The problem is PDF as "just content":**
- "ebook" at $4.99 = commodity
- "ebook" at $47 that "solves a $10,000 problem" = serious buyers
- Positioning matters: "PDF ebook" ($5-15 perceived), "interactive toolkit with decision frameworks" ($49-149 perceived)

## The Gap in the Market

1. **Infrastructure Decision Frameworks** -- Nobody sells an interactive tool for decisions like "ECS vs EKS vs Lambda for this workload." ByteByteGo does system design; nobody does infrastructure choices.
2. **Production Infrastructure Teardowns** -- Real architectures with decision rationale explained.
3. **Curated starter kits with educational READMEs** -- The GDPR Terraform toolkit at $59-99 is proof of concept.
4. **Assessment/Diagnostic Tools** -- "What's your infrastructure maturity?" Google's DORA proves demand; nobody productized at indie level.

## Recommended Format: "Decision Kit"

The highest-converting format at $29-79 combines:
- **Structured digital guide** (GitBook or well-designed web format, NOT raw PDF)
- **Companion GitHub repo** with real infrastructure code + decision-annotated READMEs
- **Interactive decision framework** (even a simple web-based flowchart/quiz)
- **Templates/boilerplates** that are immediately deployable

**Why this wins:**
- Hands-on (GitHub repo with real code)
- Decision-focused (what DevOps people actually struggle with)
- Updatable (not a static PDF)
- Immediately useful (templates they can deploy today)
- Premium-positioned (not "another ebook" -- it's a "decision toolkit")
