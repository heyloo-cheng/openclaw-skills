# Narrative Pass

### Don't present data. Introduce a person. Make the audience become that person. Then show them the gap between where they are and where they could be.

## What It Does

Takes an existing slide deck (or landing page, or pitch document) and applies a persona-anchored storytelling layer. Transforms data presentations into stories that create neural engagement.

```
Existing deck (data dump)
    ↓
Step 1: Choose anchor persona from pipeline data
    ↓
Step 2: Map SPIN arc (Situation → Problem → Implication → Need-Payoff)
    ↓
Step 3: Beat map (emotional sequence with tension curve + TURN)
    ↓
Step 4: Write transition lines (persona callbacks, questions, delivery cues)
    ↓
Step 5: SPIN check every slide (whose story? what's the gap? what's the question?)
    ↓
Step 6: Rewrite speaker notes as performance scripts
    ↓
Output: Narrative-layered deck with speaking script
```

## When to Use

- After `slidev-deck` generates a deck (Phase 4.5)
- After `landing-page` generates copy (as a narrative review pass)
- Before any pitch meeting (to write the speaking script)
- When a deck "has all the right slides but doesn't tell a story"

## The Science

Displayed founder passion → neural engagement → investor interest. This isn't opinion — it's fMRI data (Warnick et al., 2018). Named characters create 2.5x more neural engagement than abstractions. Circular stories are 22% more memorable than linear ones.

The narrative pass operationalizes this research into a repeatable skill.

## Key Concepts

| Concept | Source | Application |
|---------|--------|-------------|
| **Anchor Persona** | Persona-extract pipeline | Named protagonist drawn from real quotes |
| **SPIN Arc** | Neil Rackham, SPIN Selling | Situation → Problem → Implication → Need-Payoff |
| **Gap Selling** | Keenan | Every slide shows current state vs. future state |
| **Tension Curve** | Aegir outline-writer | Build tension first half, TURN at midpoint, conviction second half |
| **Neural Engagement** | Warnick et al. (2018) | Passion + specificity → mirror neuron activation |
| **Circular Narrative** | Stanford research | Closing callbacks to opening = 22% more memorable |
